2. What the App does

Mnemonik is a memory-training app. Core training can work offline on your device. Cloud features (sync, leaderboards, duels, community chat, notifications) use Google Firebase.

3. Data we collect

3.1 Account data

  • Email address and password (email sign-up).
  • One-time email verification codes (processed server-side via Cloud Functions).
  • Anonymous user ID (guest sign-in).
  • Display name, optional profile photo, optional “about me” text.
  • Firebase Authentication user ID (UID).

Note: Google Sign-In may be enabled in a future release; Google may share name, email, and profile photo per your Google account settings.

3.2 Usage & progress

  • Training scores, streaks, per-mode statistics, session history.
  • Quest and learning progress; app settings (language, theme, etc.).
  • Leaderboard points and aggregated public stats (if sharing is enabled).

3.3 Social & multiplayer

  • Community chat: message text, display name, user ID, timestamps; images may be posted by authorized admins only.
  • Duels: room state, scores, in-room chat, invites.
  • Online presence (last activity timestamp).

3.4 Notifications

  • FCM push token; device time zone for local reminders; activity data for smart reminders.

3.5 On-device data

  • Photos/files you explicitly pick for training (faces mode, loci routes, imports). Access requires your permission.
  • Local storage (SharedPreferences) for progress and settings.

Face-training image bytes are not uploaded to the cloud—only metadata without images is synced.

3.6 What we do not collect

  • Precise GPS location; contacts/SMS/call logs.
  • Advertising ID (no ads in the current version).
  • Payment data (in-app purchases not connected yet).
  • Firebase Analytics / Crashlytics (not used in the current app code).
